About Clearoid

Clearoid is an AI-powered semantic duplicate detection system designed to identify duplicate and similar project titles using advanced Natural Language Processing techniques.

Unlike traditional keyword-based systems, Clearoid uses embedding-based semantic similarity to understand the meaning behind text, allowing it to detect duplicates even when wording differs.

Clearoid helps:

  • Students avoid duplicate project topics
  • Institutions maintain unique project databases
  • Organizations ensure clean and accurate data

Key Features:

  • Real-time duplicate detection
  • Bulk Excel upload support
  • AI-based semantic similarity
  • Duplicate clustering
  • Submission history tracking

Technology Stack:

  • Frontend: React.js / Next.js / Tailwind CSS
  • Backend: Node.js API
  • Database: Supabase PostgreSQL
  • AI Engine: Transformer-based embedding model
